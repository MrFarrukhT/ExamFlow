# 🎤 Cambridge Speaking Test System

## Quick Start

### For Invigilators/Admin

1. **Start the server:**
   - Double-click `start-speaking-system.bat`
   - OR run: `node cambridge-database-server.js`

2. **Access evaluation dashboard:**
   - Open `cambridge-speaking-evaluations.html` in your browser
   - Or click "🎤 Speaking Evaluations" from admin dashboard

3. **Evaluate submissions:**
   - Click on any pending submission
   - Listen to the recording
   - Score on all criteria
   - Save evaluation

### For Students

1. **Login to Cambridge system:**
   - Use your student ID and name
   - Select your Cambridge level

2. **Access speaking test:**
   - Navigate to Speaking Test from your level folder
   - Example: `Cambridge/MOCKs-Cambridge/A2-Key/speaking.html`

3. **Complete test:**
   - Test your microphone first
   - Start recording when ready
   - Answer all questions
   - Review and submit

## 📂 Speaking Test Locations

- **A1 Movers**: `Cambridge/MOCKs-Cambridge/A1-Movers/speaking.html`

## 🎯 Features

✅ Microphone quality testing
✅ Visual recording indicators
✅ Pause and resume recording
✅ Playback before submission
✅ Re-record option
✅ Automatic database saving
✅ Invigilator evaluation dashboard
✅ Detailed scoring criteria
✅ Download recordings
✅ Filter and search submissions

## ⚠️ Requirements

- Node.js installed
- Database server running (port 3003)
- Modern browser (Chrome, Firefox, Edge)
- Microphone connected

## 🔧 Troubleshooting

**Problem**: Microphone not working
- Check browser permissions
- Allow microphone access
- Test in system settings

**Problem**: Can't submit recording
- Check server is running
- Verify internet connection
- Try refreshing page

**Problem**: Can't see submissions
- Start database server
- Check `cambridge-speaking-evaluations.html`
- Clear filters
