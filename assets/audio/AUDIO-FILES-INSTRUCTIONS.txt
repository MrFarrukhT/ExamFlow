AUDIO FILES DIRECTORY - Cambridge A2 Key Listening Test
===================================================================

This directory should contain the audio files for the A2 Key Listening test.

REQUIRED FILES:
---------------
Place the following MP3 audio files in this directory:

1. a2-key-listening-part1.mp3  (Part 1: Questions 1-5)
   - Short conversations and announcements
   - Approximately 2-3 minutes
   
2. a2-key-listening-part2.mp3  (Part 2: Questions 6-11)
   - Telephone conversation about sports club
   - Note completion task
   - Approximately 3-4 minutes
   
3. a2-key-listening-part3.mp3  (Part 3: Questions 12-16)
   - Conversation between two friends about holiday
   - Approximately 4-5 minutes
   
4. a2-key-listening-part4.mp3  (Part 4: Questions 17-21)
   - Teacher talking about school trip
   - Monologue format
   - Approximately 3-4 minutes
   
5. a2-key-listening-part5.mp3  (Part 5: Questions 22-26)
   - Five short conversations
   - Matching task
   - Approximately 3-4 minutes

AUDIO SPECIFICATIONS:
---------------------
- Format: MP3
- Quality: 128 kbps or higher recommended
- Sample Rate: 44.1 kHz
- Channels: Mono or Stereo
- Clear audio with minimal background noise
- British or American English accent (native speakers)

RECORDING TIPS:
---------------
1. Follow Cambridge A2 Key exam format guidelines
2. Use natural, conversational English
3. Speak at a moderate pace (not too fast)
4. Include appropriate pauses between questions
5. Ensure audio quality is clear and professional
6. Test audio files before deployment

TESTING:
--------
Before using in production:
1. Play each audio file to verify quality
2. Check that audio length matches question time requirements
3. Verify all questions can be answered from the audio
4. Test in different browsers to ensure compatibility
