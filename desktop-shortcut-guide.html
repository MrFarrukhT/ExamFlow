Create Desktop Shortcut

Quick access to your Test System

1 Find the Test System File

Navigate to your test system folder and locate the main file:

index.html
📍 Current Location:
C:\Users\Windows 11\Desktop\Scalable Architecture\Test System(v2)\index.html
2 Right-Click the File

Right-click on index.html to open the context menu

3 Create Shortcut

Select "Create shortcut" from the menu

Windows will create a shortcut file with the same icon

4 Move to Desktop

Drag the shortcut file to your Desktop

Or right-click the shortcut → Cut → Go to Desktop → Paste

5 Rename (Optional)

Right-click the desktop shortcut → Rename

Suggested names:

  • Innovative Centre Test System
  • Innovative Centre
  • Practice Test

Your Desktop Shortcut Will Look Like:

Innovative Centre Test System

🎯 Quick Access Tips:

  • Pin to Taskbar: Right-click shortcut → "Pin to taskbar"
  • Pin to Start Menu: Right-click shortcut → "Pin to Start"
  • Keyboard Shortcut: Right-click shortcut → Properties → Shortcut key
