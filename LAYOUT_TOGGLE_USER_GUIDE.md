# Layout Toggle - User Guide

## What is Layout Toggle?

Layout Toggle is a feature that allows you to quickly change how the reading passage and questions are displayed on your screen. You can make the reading passage bigger, make the questions bigger, or hide one side completely to focus on what you're working on.

## For IELTS Reading Tests

### How to Use:
1. **Find the Resizer**: Look for the vertical bar in the middle of your screen that separates the reading passage (left) from the questions (right). It has a small icon with arrows.

2. **Double-Click to Toggle**: **Double-click** (click twice quickly) on the resizer handle. The layout will change immediately.
   - **Important**: Use double-click, not single-click (single-click is for dragging)

3. **Cycle Through Modes**: Each double-click switches to the next layout:
   - Double-click 1: **50/50 Split** → **Reading Focus** (70/30)
   - Double-click 2: **Reading Focus** → **Questions Focus** (30/70)
   - Double-click 3: **Questions Focus** → **Full Reading** (95/5)
   - Double-click 4: **Full Reading** → **Full Questions** (5/95)
   - Double-click 5: **Full Questions** → **50/50 Split** (back to start)

4. **See What Mode You're In**: A small message will pop up at the bottom of your screen showing the current layout name.

### Layout Modes Explained:

#### 🔄 50/50 Split (Default)
- Equal space for reading and questions
- Good for: Moving back and forth between passage and questions
- Best when: You need to reference the passage frequently

#### 📖 Reading Focus (70/30)
- More space for the reading passage
- Questions still visible but smaller
- Good for: First read-through of the passage
- Best when: You want to read carefully without scrolling too much

#### ✍️ Questions Focus (30/70)
- More space for questions
- Passage still visible for quick reference
- Good for: Answering questions that don't need constant passage reference
- Best when: Working on multiple-choice or true/false questions

#### 📚 Full Reading (95/5)
- Almost full screen for reading passage
- Questions minimized to a thin strip
- Good for: Detailed reading without distractions
- Best when: Making notes or highlighting important parts

#### 📝 Full Questions (5/95)
- Almost full screen for questions
- Passage minimized to a thin strip
- Good for: Focusing on question structure
- Best when: You've finished reading and are answering questions

### Tips:
- You can still **drag** the resizer to manually adjust the size (single-click and drag)
- The toggle works **in addition** to manual dragging
- **Double-click** for quick layout changes, **single-click + drag** for precise adjustments
- Layout changes are **smooth and animated**
- All your answers are **automatically saved**
- You can toggle at **any time** during the test

## For Cambridge Tests

### How to Use:
1. **Find the Button**: Look in the top-right corner of your screen for a button with a **columns icon** (looks like two rectangles side by side: ▐▐)

2. **Click to Toggle**: Click the button once to change the layout

3. **Watch the Icon**: The button icon changes to show what layout you're in:
   - ▐▐ Columns = Side by side
   - ≡ Align left = Reading focus
   - ≡ Align right = Questions focus
   - 📄 Document = Full reading
   - ☰ List = Full questions

4. **See Confirmation**: A message appears at the bottom showing the current layout

### Layout Modes:

#### 📊 Side by Side (Default)
- Reading material and questions next to each other
- Natural layout for Cambridge tests
- Use when: You need both visible equally

#### 📖 Reading Focus
- More room for reading material (70%)
- Questions smaller but still visible (30%)
- Use when: Reading longer passages

#### ✍️ Questions Focus  
- More room for questions (70%)
- Reading material smaller (30%)
- Use when: Focusing on answering

#### 📄 Full Reading
- Reading material takes almost full width
- Questions hidden
- Use when: Detailed study of images or text

#### 📝 Full Questions
- Questions take almost full width
- Reading material hidden
- Use when: You remember the passage content

### Special Note:
- Layout toggle **only appears** on pages with reading passages
- Writing-only pages (Parts 6 & 7) won't show the button
- Your layout choice **resets** when switching between parts

## Common Questions

**Q: Will this affect my test score?**
A: No, it's just a display feature. Your answers are saved no matter which layout you use.

**Q: Can I use keyboard shortcuts?**
A: Currently, you need to click the resizer or button. Keyboard shortcuts may be added in the future.

**Q: What if I accidentally click while trying to drag?**
A: The system is smart - it only toggles if you click quickly. Dragging won't trigger the toggle.

**Q: Does my layout choice save for next time?**
A: Currently, it resets to 50/50 when you reload. We may add a memory feature in the future.

**Q: Can I customize the percentages?**
A: Not yet, but we're considering adding this in a future update.

**Q: Does this work on tablets?**
A: Yes! Touch screens are fully supported.

## Troubleshooting

**Problem: The resizer doesn't toggle when I click**
- Make sure you're **double-clicking** (clicking twice quickly)
- Try double-clicking on the center of the resizer handle (the middle bar with the icon)
- Click fast enough - both clicks should be within about half a second
- If you're dragging, release first then double-click

**Problem: I don't see the layout button in Cambridge tests**
- The button only appears on pages with reading passages
- Check that you're on Parts 1-5 (not 6 or 7)
- Try refreshing the page

**Problem: The layout looks strange or broken**
- Try clicking through all 5 modes to reset
- Refresh the page to return to default
- Check your browser zoom level (should be 100%)

**Problem: Toast message doesn't disappear**
- It should auto-disappear after 2 seconds
- If stuck, refresh the page
- This might be a browser issue - try a different browser
